# 🦀 ZeroClaw Agent 使用指南

欢迎使用 ZeroClaw！这是你智能家居的最强大脑。本插件运行在极致轻量的 Daemon 守护模式下，并为你提供了一个极客级的 Web 终端。

## 🛠️ 1. 基础配置说明
在插件的“配置”选项卡中，你可以设置基础参数：
* **provider**: 默认调用的模型平台（如 `nvidia`, `groq`, `openrouter`, `openai` 等）。
* **port**: 网关对外提供 Webhook 的端口（默认 `8080`）。

> **⚠️ 高级安全警告**：
> 真正的 API 密钥请**不要**直接写在界面的配置框里！为了绝对安全，请通过 Web 终端配置本地 `.env` 文件。

## 💻 2. 进入极客控制台 (Web Terminal)
本插件内置了 `ttyd` 终端。
1. 启动插件后，点击界面上的 **打开 Web UI**（或者访问你配置的 `8099` 端口）。
2. 你将直接进入 ZeroClaw 的内部环境。

## 🔑 3. 配置多模型武器库 (高级玩家必看)
想要体验满血版的多模型切换，请在 Web 终端中执行以下操作：
编辑环境变量隐藏文件：
   ```bash
   vi /config/.zeroclaw/.env
   ```
## 🩺 4. 故障排查 (Doctor自检)
如果发现飞书机器人没理你，或者模型调用报错，请在 Web 终端中输入：
    ```bash
    zeroclaw doctor
    ```
